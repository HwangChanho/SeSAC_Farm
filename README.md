# SeSAC Farm (1.0 ver)

## 평가과제 (~ 1/6)

디자인 패턴 : MVVM
FrameWork : UIkit
개발기간: 이틀

보완사항

- post 뷰에 스크롤뷰 추가
- 댓글 textfield가 댓글 가림현상 수정
- RxSwift로 구조 변경 (일부)
- 로그아웃 기능 추가
